
Q1: Why might we consider using Git?

Answer: We might consider using Git since it allows multiple people to work simultaneously on a project and  manage and track changes in files 
	in order to go back to previous versions, compare changes, and recover lost work.

	including their probability, symptoms, causes, and potential outcomes.


Q3: What is the difference between Merging and Rebasing?

Answer: Merging takes the changes from the feature branch and applies them to the current branch while keeping the history of both branches. 
	On the other hand, rebasing moves the changes from one branch to another branch while rewriting the history.
